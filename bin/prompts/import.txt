Сформируй JSON-сводку текущего диалога.

ВАЖНО: Выдай JSON одним блоком кода (```json ... ```). Перед блоком кода добавь строку "COPY_JSON_BELOW", после блока — строку "END_JSON".

Используй структуру:

{
  "title": "<краткий заголовок диалога, 5-8 слов>",
  "summary": "<одно-два предложения, передающие суть всего диалога>",
  "turns": [
    {
      "request": "<текст вопроса пользователя>",
      "response": "<текст ответа ассистента>",
      "tags": ["<тег1>", "<тег2>", "<тег3>"]
    }
  ]
}

Требования:
- Теги: только строчные латинские буквы, цифры, дефисы (slug-формат). Минимум 1, максимум 5 на пару.
- В response включай полный текст ответа.
- В summary укажи 1-2 предложения, отражающие основную тему и ключевые выводы диалога.
- Поля request, response и tags обязательны.
- Включай в turns все пары вопрос-ответ текущего диалога.
