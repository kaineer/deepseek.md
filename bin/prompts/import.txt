Сформируй JSON-сводку текущего диалога в строгом формате, который я укажу ниже.

ВАЖНО: Выдай JSON одним блоком кода (```json ... ```). Перед блоком кода добавь строку "COPY_JSON_BELOW", после блока — строку "END_JSON". Это позволит точно определить границы данных.

Используй следующую структуру:

{
  "uuid": "<сгенерируй UUID v4 для сессии>",
  "title": "<краткий заголовок диалога, 5-8 слов>",
  "summary": "<одно-два предложения, передающие суть всего диалога>",
  "created": "<ISO 8601 timestamp первого сообщения>",
  "updated": "<ISO 8601 timestamp последнего сообщения>",
  "turns": [
    {
      "uuid": "<UUID v4 для этой реплики>",
      "timestamp": "<ISO 8601 timestamp вопроса>",
      "request": "<текст вопроса пользователя>",
      "response": "<текст ответа ассистента>",
      "tags": ["<тег1>", "<тег2>", "<тег3>"]
    }
  ]
}

Требования:
- Теги: только строчные латинские буквы, цифры, дефисы (slug). Минимум 1, максимум 5 на реплику.
- Все поля обязательны.
- В summary укажи 1-2 предложения, отражающие основную тему и ключевые выводы диалога.
- В response включай полный текст ответа.
